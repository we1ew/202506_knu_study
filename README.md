## 사용 권한

본 프로젝트는 2025 개인정보 보안증적 프로그램 개발 교육의 구성원들에게만 사용이 허용됩니다. 깃헙 레포지토리 자체는 공개 상태이나 외부인 또는 비구성원의 사용시 어떠한 피드백도 제공하지 않습니다.
